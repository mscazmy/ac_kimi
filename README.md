# بريز كول - خدمات تكييف الهواء في واسط

## نظرة عامة
موقع إلكتروني لخدمات تكييف الهواء في محافظة واسط، العراق. يتضمن صفحة رئيسية ولوحة تحكم للإدارة.

## المميزات
- **تصميم عصري** بألوان السماء الزرقاء
- **تصميم متجاوب** يعمل على جميع الأجهزة
- **معرض صور** تفاعلي مع إمكانية التكبير
- **نموذج تواصل** لطلبات الخدمة
- **لوحة تحكم** محمية بكلمة مرور
- **إدارة المنشورات** (إضافة، تعديل، حذف)
- **إدارة الصور** (إضافة، حذف)
- **إدارة طلبات التواصل**
- **تصدير البيانات** لـ JSON
- **جاهز للربط** بـ Supabase في المستقبل

## كيفية النشر على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub
2. ارفع جميع الملفات إلى المستودع
3. اذهب إلى Settings > Pages
4. اختر الفرع (branch) الرئيسي ومجلد الجذر
5. انتظر قليلاً ثم ستظهر رابط الموقع

## بيانات الدخول للوحة التحكم
- **اسم المستخدم:** `admin`
- **كلمة المرور:** `admin123`

## ملاحظات مهمة
- الصورة الرئيسية (hero-banner.png) تحتوي على نصوص مدمجة، لذلك يتم عرضها بشكل منفصل دون أي تداخل
- جميع البيانات تُحفظ في localStorage حالياً
- يمكن استبدال الصورة الرئيسية بسهولة من إعدادات لوحة التحكم
- الموقع جاهز للربط بـ Supabase Authentication لاحقاً
